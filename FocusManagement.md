# Focus Management

## Deep Work
### What is Deep Work?
- Working with full focus
- No distractions
- Mentally demanding work
- Produces high-quality results
- Improves skill and productivity

## Summary of Deep Work Book
### How to do deep work properly?
- Schedule time for deep work
- Remove distractions
- Work in focused blocks
- Set clear goals for each session
- Practice regularly

### How can you implement these principles in daily life?
- Fix a daily deep work time
- Keep phone away while working
- Work on one task at a time
- Use short breaks between sessions
- Avoid social media during work hours

## Dangers of Social Media
### Cons of social media
- Reduces attention span
- Increases distraction
- Causes addiction
- Lowers productivity
- Affects mental health
